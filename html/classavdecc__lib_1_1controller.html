uint32_t  talker_capabilities_flags,
uint32_t  listener_capabilities_flags 
)
pure virtual

Apply filters required to be true for an end station to be enumerated.

The required filters are passed in. If any of the required entity, talker, or listener flags are not true for an end station, the end station is not enumerated.
